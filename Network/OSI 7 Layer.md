### `Application layer`
* Related to real applications like chrome, safari, & so on..
* in TCP/IP) Application protocol

### `Presentation layer`
* Make application data to suitable data for communicating.
* ex) Data Encryption & Decryption

### `Session layer`
* Make sessions.
* Make settings for connections & disconnections
* Make logical connection.

**Above layers are upper layer for data communication.** <br>
**Protocol : Application**

****************************************

### `Transport layer`
* Communicate real data.
* Handle the size, speed, object.
* Protocol : `TCP`, `UDP`

### `Network Layer`
* Send packets
* Most routers are this layer.
* Protocol : `IP`

### `Data link layer`
* Provide data communications between nodes
* Handle errors caused by physical layer
* MAC LLC
* Most switches are this layer.
* Protocol : `Ethernet protocol`

### `Physical layer`
* Real physical. Literal meaning.
* Protocol : `Ethernet protocol`
